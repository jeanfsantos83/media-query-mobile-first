Conhecendo Media Queries em CSS

Consultas de mídia são um recurso do CSS 3 que permite que a renderização do conteúdo se adapte a diferentes condições, como resolução da tela (por exemplo, tamanho da tela do celular e do desktop). Tornou-se um padrão recomendado pelo W3C em junho de 2012, e é uma tecnologia fundamental de web design responsivo (RWD). As consultas de mídia foram esboçadas pela primeira vez na proposta inicial de CSS de Håkon Wium Lie em 1994, mas não se tornaram parte do CSS 1.

A Recomendação HTML4 de 1997 mostra um exemplo de como as consultas de mídia poderiam ser adicionadas no futuro. Em 2000, o W3C começou a trabalhar em consultas de mídia e também em outro esquema para suporte a vários dispositivos: CC/PP . Os dois abordam o mesmo problema, mas o CC/PP é centrado no servidor, enquanto as consultas de mídia são centradas no navegador. O primeiro rascunho de trabalho público para consultas de mídia foi publicado em 2001. Media Queries Nível 3, publicado como uma recomendação candidata em 27 de julho de 2010, tornou-se uma recomendação do W3C em 19 de junho de 2012. As correções propostas foram publicadas em 5 de abril. 2022.

Media Queries Level 4, publicado como um rascunho de trabalho em 9 de maio de 2017, era um rascunho de recomendação de candidato do W3C em 25 de dezembro de 2021.

Uma consulta de mídia consiste em um tipo de mídia e uma ou mais expressões, envolvendo recursos de mídia , que são resolvidas como verdadeiras ou falsas. O resultado da consulta será verdadeiro se o tipo de mídia especificado na consulta de mídia corresponder ao tipo de dispositivo no qual o documento está sendo exibido e todas as expressões na consulta de mídia forem verdadeiras. Quando uma consulta de mídia é verdadeira, a folha de estilos ou regras de estilo correspondentes são aplicadas, seguindo as regras normais em cascata. Consultas de mídia usam o CSS "at-rule".

Exemplos:

A seguir estão exemplos de consultas de mídia CSS:

@ media screen and ( display-mode : fullscreen ) { /* O código aqui se aplica apenas a telas em tela cheia */ }

@ media all and ( orientation : Landscape ) { /* O código aqui só se aplica na orientação paisagem */ }

@ media screen and ( min-device-width : 500px ) { /* O código aqui se aplica apenas a telas iguais ou maiores que 500 pixels de largura */ }

Tipos de mídia

Um tipo de mídia pode ser declarado no cabeçalho de um documento HTML usando o atributo “media” dentro de um elemento. O valor do atributo "media" especifica em qual dispositivo o documento vinculado será exibido. O tipo de mídia “todos” também pode ser usado para indicar que uma folha de estilo se aplica a todos os tipos de mídia.
